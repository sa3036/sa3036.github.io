Organize contents of research paper in latex files

Organize contents of your research paper into the followings latex files(.tex)

  • paper.text
    • This file should contain the title of your paper, the headers, and inclusion of the files that you need for your paper (as below).
  • macro.tex
    • This file should contain all packages, commands, etc. (just copy from the template)
  • abstract.tex
  • intro.tex
  • background.tex
  • Approach.tex
  • results.tex
  • related.tex
  • conclusion.tex
  • 00.bib
    • This file should contain the references to research papers, journals, etc.
  • Make file
    • A make file to build the latex package (just copy from the template). Please refer to the template from repo Research paper template". Run "make" before you commit and make sure you paper compiles every time you commit.
